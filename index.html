Your Anthropic API key

Punslinger calls the Claude API directly from your browser using a key you provide. Nothing is sent to us — your key is stored only in this browser's local storage, and you're billed directly by Anthropic for what you generate.

Don't have a key yet?

  1. Go to console.anthropic.com and sign up or log in.
  2. Add billing credit (a few dollars covers a lot of runs).
  3. Create an API key and paste it below.

This is a "bring your own key" tool: the cost of every run is yours, paid directly to Anthropic — not a subscription, not a markup.
